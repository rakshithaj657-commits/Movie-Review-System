In the Movie Review System context:
- Application UI: Web interface for users to submit/view reviews
- Spark SQL/DataFrames: Used for querying movie metadata and reviews
- Spark Streaming: Processes real-time review submissions
- MLlib: Performs sentiment analysis on reviews
- GraphX: Analyzes user relationships and social connections
- Spark Core: Distributes processing tasks across the cluster
